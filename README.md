# Nar23_C-Space-Invader
This was a project done during the training for Bina Nusantara University Blue Jacket (Lab Assistant) 2022. The task was to replicate a premade game with many features and doing so by using certain algorithms and data structures

# How to open
The game can be played through the "game.exe" file. The .exe can also be compiled from "main.cpp"

# Controls

Navigate the menu with "W" and "S", and use "Enter" to select


### Basic Movement
<ul>
<li>'w' - Move U</li>
<li>'a' - Move Left</li>
<li>'s' - Move Down</li>
<li>'d' - Move Right</li>
</ul>

### Lobby Arena
<ul>
<li>'SPACE' - Interact</li>
<li>'o' - Open backpack</li>
</ul>

### Game Arena
<ul>
<li>'SPACE' - Shoot</li>
<li>'r' Reload</li>
<li>'f' Skill</li>
<li>'1' Potion</li>
<li>'2' Max Potion</li>
<li>'3' Energy Drink</li>
<li>'4' Max Energy Drink</li>
<li>'g' Use Bomb</li>
</ul>
